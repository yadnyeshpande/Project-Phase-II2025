Skin Lesion Analysis

Early Detection of Skin Diseases Using a Deep Learning-Based Image Classification Model

About This Project

This machine learning-based system analyzes skin lesion images and classifies them into different disease categories. It is built using deep learning techniques to assist dermatologists and individuals in the early detection of melanoma and other skin conditions. Our model is trained on the HAM10000 dataset to achieve high accuracy and reliability in skin disease classification.
